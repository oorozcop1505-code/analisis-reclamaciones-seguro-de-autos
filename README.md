# auto-insurance-claims-analysis
Análisis exploratorio en reclamaciones de seguros de auto, con un enfoque en Fraudes y severidad de los siniestros 

## Objetivo
Analizar reclamaciones de seguros de auto para identificar patrones asociados a la severidad de los siniestros y la detección de fraude, utilizando SQL y herramientas de visualización, con un enfoque en toma de decisiones basada en datos.

## Dataset
Conjunto de datos sintéticos de reclamaciones de seguros de autos.

## Herramientas
- SQL (MySQL)
- Excel (exploración inicial)
- Power BI (Visualización)
- GitHub (documentación)
- Temas clave
- Severidad y costos de las reclamaciones
- Detección de fraudes
- Patrones demográficos y geográficos
- Validación de la calidad de los datos
